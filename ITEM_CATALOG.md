## Item Schema

All items in `fishing_items.json` follow this structure:

```json
{
  "items": {
    "unique_key": {
      "type": "trash|legendary_component|gift|consumable",
      "key": "unique_key",
      "name": "Display Name with Emoji",
      "emoji": "ğŸ”®",
      "sell_price": 0,
      "description": "Item description",
      "effect_type": "optional_effect",
      "effect_value": 0.9,
      "original_value": 0.65,
      "one_time_use": true
    }
  }
}
```

### Field Definitions

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| type | string | Yes | Category: trash, legendary_component, gift, consumable |
| key | string | Yes | Unique identifier (matches JSON key) |
| name | string | Yes | Display name (includes emoji prefix) |
| emoji | string | Yes | Single emoji for UI display |
| sell_price | number | Yes | Coins earned when sold (0 = not sellable) |
| description | string | Yes | Item flavor text |
| effect_type | string | No | Type of effect (for consumables/special items) |
| effect_value | number | No | New value after effect (for consumables) |
| original_value | number | No | Original value before effect |
| one_time_use | boolean | No | Whether item is consumed on use |

## Loading Items in Code

### From constants.py

```python
from cogs.fishing.constants import TRASH_ITEMS, LEGENDARY_ITEMS, GIFT_ITEMS, ALL_ITEMS_DATA

# Use trash items
random_trash = random.choice(TRASH_ITEMS)

# Use legendary items
for item in LEGENDARY_ITEMS:
    print(f"{item['emoji']} {item['name']}")

# Use gift items (as keys)
if "cafe" in GIFT_ITEMS:
    print("Gift item available")

# Use full items data
all_item_keys = list(ALL_ITEMS_DATA.keys())
```

### From consumables.py

```python
from cogs.fishing.consumables import CONSUMABLE_ITEMS, get_consumable_info

# Check if item is consumable
item_info = get_consumable_info("nuoc_tang_luc")

# Get effect value
boost_rate = item_info["effect_value"]  # 0.90
```

## Item Loot Tables

### Fishing Loot Distribution

Items are obtained through various game mechanics:

**Normal Fishing (LOOT_TABLE_NORMAL)**:
- Trash: 30% chance
- Common Fish: 60% chance
- Rare Fish: 5% chance
- Chest: 5% chance

**Boosted Fishing (LOOT_TABLE_BOOST)**:
- Trash: 15% chance
- Common Fish: 75% chance
- Rare Fish: 5% chance
- Chest: 5% chance

**Chest Drops (CHEST_LOOT)**:
- fertilizer: 25%
- puzzle_piece: 15%
